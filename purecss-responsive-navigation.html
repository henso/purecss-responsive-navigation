This is a pure demo site

for demonstration lovers

The incredible responsive menu...

Yahoo made this nice small Pure CSS. On their demonstration site they show the surface, some examples, some code. Unfortunately, the almost totally sweet responsive sidebar navigation is no official part of it.

This little rip-off is a chopped down version of the origin YUI Pure CSS demo page, to show you the bare minimal you need to implement a toggable responsive sidebar navigation. Everything except the HEAD tag with the css resources, the navigational #menu section and the menu.js JS resource tag at the end of the page is overhead.

Tiny disclaimer at the end - I had nothing to do with the creative process making the Yahoos Pure CSS. I just poked around to figure out what makes the responsive navigation tick.
